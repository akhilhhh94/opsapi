# Cloudflare API

## Configuration

Tag: `cloudflare`
```shell
CLOUDFLARE_ZONE: 
CLOUDFLARE_RECORD_NAME: 
CLOUDFLARE_RECORD_TYPE: 
CLOUDFLARE_RECORD_VALUE: 
CLOUDFLARE_ACCOUNT_EMAIL: 
CLOUDFLARE_ACCOUNT_API_KEY:

```
